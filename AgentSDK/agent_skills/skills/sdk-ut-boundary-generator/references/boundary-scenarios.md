# 边界场景分类

本文档定义SDK单元测试中需要覆盖的边界场景分类。

---

## 1. 空值/null输入场景

### 1.1 指针类空值

| 场景编号 | 场景名称 | 说明 | 测试要点 |
|---------|---------|------|---------|
| N-001 | 空指针参数 | 指针参数传入NULL/nullptr | 验证函数正确处理空指针，不崩溃 |
| N-002 | 双重指针空值 | 二级指针参数传入NULL | 验证输出参数的正确处理 |
| N-003 | 回调函数空值 | 回调函数指针为NULL | 验证默认行为或错误处理 |

### 1.2 字符串类空值

| 场景编号 | 场景名称 | 说明 | 测试要点 |
|---------|---------|------|---------|
| N-010 | 空字符串 | 字符串参数为"" | 验证空字符串的合法处理 |
| N-011 | NULL字符串指针 | C风格字符串指针为NULL | 验证不崩溃 |
| N-012 | 仅空白字符串 | 字符串仅包含空格/制表符 | 验证trim处理 |

### 1.3 容器类空值

| 场景编号 | 场景名称 | 说明 | 测试要点 |
|---------|---------|------|---------|
| N-020 | 空数组/列表 | 数组或列表参数为空 | 验证返回空结果或正确处理 |
| N-021 | 空字典/映射 | 字典参数为空 | 验证默认值处理 |
| N-022 | 空迭代器 | 迭代器指向容器末尾 | 验证边界处理 |

### 1.4 对象类空值

| 场景编号 | 场景名称 | 说明 | 测试要点 |
|---------|---------|------|---------|
| N-030 | 空对象引用 | 对象引用为null/None | 验证异常或错误返回 |
| N-031 | 未初始化对象 | 对象未正确初始化 | 验证初始化检查 |
| N-032 | 已销毁对象 | 对象已被销毁后使用 | 验证生命周期管理 |

## 2. 边界值场景

### 2.1 数值边界

| 场景编号 | 场景名称 | 说明 | 测试要点 |
|---------|---------|------|---------|
| B-001 | 整数最大值 | INT_MAX, LONG_MAX等 | 验证溢出处理 |
| B-002 | 整数最小值 | INT_MIN, LONG_MIN等 | 验证下溢处理 |
| B-003 | 零值 | 数值参数为0 | 验证除零保护 |
| B-004 | 负值 | 参数传入负数 | 验证符号处理 |
| B-005 | 浮点特殊值 | NaN, Infinity, -Infinity | 验证特殊值处理 |
| B-006 | 浮点精度边界 | 极小浮点数、精度边界 | 验证精度处理 |

### 2.2 长度边界

| 场景编号 | 场景名称 | 说明 | 测试要点 |
|---------|---------|------|---------|
| B-010 | 字符串最大长度 | 达到最大允许长度 | 验证缓冲区处理 |
| B-011 | 字符串超长长度 | 超过最大允许长度 | 验证截断或错误 |
| B-012 | 数组最大容量 | 达到容器最大容量 | 验证扩容或错误 |
| B-013 | 单字符字符串 | 字符串长度为1 | 验证边界处理 |

### 2.3 索引边界

| 场景编号 | 场景名称 | 说明 | 测试要点 |
|---------|---------|------|---------|
| B-020 | 首元素索引 | 索引为0 | 验证正确访问 |
| B-021 | 末元素索引 | 索引为length-1 | 验证正确访问 |
| B-022 | 越界索引 | 索引为length或更大 | 验证错误处理 |
| B-023 | 负索引 | 索引为负数 | 验证错误处理或反向索引 |

### 2.4 容量边界

| 场景编号 | 场景名称 | 说明 | 测试要点 |
|---------|---------|------|---------|
| B-030 | 缓冲区满 | 缓冲区刚好填满 | 验证边界写入 |
| B-031 | 缓冲区溢出 | 写入超过缓冲区容量 | 验证保护机制 |
| B-032 | 队列满 | 队列达到最大容量 | 验证阻塞或错误 |
| B-033 | 栈满/栈空 | 栈的边界状态 | 验证边界操作 |

## 场景优先级

| 优先级 | 场景类型 | 说明 |
|--------|---------|------|
| P0 | 空值/null输入 | 最常见的边界问题 |
| P0 | 数值边界 | 容易导致安全问题 |
| P1 | 非法输入 | 重要的输入验证 |
| P1 | 索引边界 | 常见的越界问题 |
| P2 | 资源限制 | 需要环境配合 |
| P2 | 状态错误 | 需要理解业务逻辑 |
| P3 | 时序相关 | 难以稳定复现 |
| P3 | 兼容性相关 | 需要多环境测试 |
