- **SDK入口数**：[已识别的入口数量]

## 测试文件建议

### 文件命名

| 语言 | 命名规范 | 示例 |
|------|---------|------|
| C++ | `<源文件名>_test.cpp` | `api_test.cpp` |
| Python | `test_<模块名>_boundary.py` | `test_api_boundary.py` |

### 文件位置

```
project/
├── docs/                    # API文档目录
│   └── [搜索到的文档]
├── src/
│   └── module/
│       └── api.cpp
└── test/                    # 或 tests/
    └── module/
        └── api_test.cpp
```
